Online Exam Management System

Spring Boot MVC – University Project

Overview

The Online Exam Management System is a web-based application developed as part of a university software engineering project.
It follows the Model–View–Controller (MVC) architectural pattern and is implemented using Java Spring Boot with server-side rendered views.

The system allows different user roles to interact with the platform, including administrators, instructors, and students, to manage and participate in online examinations.

Project Objectives

Apply the MVC architectural pattern in a real-world web application.

Implement a structured backend using Spring Boot and JPA.

Provide a simple and functional online examination workflow.

Demonstrate core software engineering concepts such as separation of concerns, layered architecture, and basic CRUD operations.
